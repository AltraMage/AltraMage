#### Lesson 1 - What is a Personal Computer (PC)
Anything that is an electronic device that takes inputs and processes it to create an output. This includes tablets, phone, consoles, calcs, servers, etc.  
Basic parts of a computer: Input, Processor, Output, Storage. Know the roles of a device and how to upgrade to improve performance. Identify difference types of storage and peripherals.

Types of software include: System software, application software, and device driver software. We will be using Cisco Packet Tracer and Vitrualbox

Safety around computers i.e. Electricity & ESD, Weight (not highly relevant), Cables (trip hazard, not across pathways), Tools.

Electricity. NZ is 230v @50hz. ESD: 3000V can be felt, 6000v Can be heard, 8000v can be seen. Components can be damaged at ~10-30v. Discharge yourself by touching metal case or PSU. New products are shipped in anti-static bags.

Safe disposal of electronics. Take appropriate measures

A command line interface (CLI) is commonly referred to as a terminal is where you manually write commands for the computer to run.  
A graphical user interface (GUI) is anything with mouse control, usally has buttons.

#### Lesson 1 - Overview of Information Systems (IS)
IS is a set of interrelated components that compile and work with our data.
This can be used to increase revenues and reduce costs
##### I/O in IS
Input is user data, i.e. names & emails.  
Process is the verification of this data, is it real and original? Also does any calcs such as trends and storage of data.
Output is user / db feedback, Success messages, data insights, etc 

Valuable information qualities:
Accurately - Is it true. Relevant - Does it matter. Timeliness - Is it given in time that makes it helpful. Completeness - Is it the full story. Clarity - is it concise and to the point.

##### IS Performance and Standards.
Consider this criteria when using or making a system.
1. Reliability - Works consistently without failure or built-in redundancy.
2. Availability - Works *when* the user needs it to.
3. Responsiveness - How quick the system reacts.
4. Throughput - Can handle vast quantities of data effectively.
5. Quality - Correct information.
6. Scalable - can be grown or shrunk as needed without impeding functions.
7. Security - Protects information and allow access only to specific people.

Computer-based information systems is a single set of hardware, software, databases, telecommunications, people, and procedures that are configured to collect, manipulate, store and process data into information.
